# <a name="export-diagrams-as-images"></a>Esportare diagrammi come immagini

In Visual Studio, è possibile salvare diagrammi di modellazione in formati di sola lettura, per poterli condividere con parti interessate, utenti o membri del team di progetto che non usano la stessa versione di Visual Studio.

Per informazioni su quali edizioni di Visual Studio questa funzionalità, vedere Supporto dell'edizione per gli strumenti di [architettura e modellazione](../modeling/analyze-and-model-your-architecture.md#VersionSupport).

## <a name="copy-a-diagram-or-part-of-a-diagram-to-another-application"></a>Copiare un diagramma o una parte di un diagramma in un'altra applicazione

1. Nel diagramma selezionare alcune o tutte le forme.

2. Fare clic con il pulsante destro del mouse sulla selezione e quindi **scegliere Copia**.

3. Incollare la selezione nell'applicazione scelta.

## <a name="save-a-modeling-diagram-as-an-xml-paper-specification-xps-file"></a>Salvare un diagramma di modellazione come XML Paper Specification file con estensione xps

1. Scegliere **Stampa** dal menu File **per** aprire la **finestra di dialogo** Stampa.

2. **Nell'elenco Nome** **selezionare** Microsoft XPS Document Writer e quindi fare clic su **OK.**

3. Nella finestra **di dialogo Salva file con** nome salvare il file come file di documento **XPS \* (con estensione xps).**
