# <a name="options-text-editor-f-intellisense"></a>Opzioni, Editor di testo, F#, IntelliSense

Usare la pagina delle opzioni **IntelliSense** per modificare alcune impostazioni dell'editor di testo per F#. Per accedere a questa pagina delle opzioni, scegliere **Opzioni** strumenti e quindi Editor  >   **di**  >  **testo F#**  >  **IntelliSense**.

## <a name="completion-lists"></a>Elenchi di completamento

- **Visualizzare l'elenco di completamento dopo la digitazione di un carattere**

   Quando questa opzione è selezionata, IntelliSense visualizza automaticamente l'elenco di completamento quando si inizia a digitare. Se non si seleziona questa opzione, il completamento intelliSense è ancora disponibile dal menu IntelliSense o premendo  +  **CTRL+BARRA SPAZIATRICE.**

- **Mostra elenco di completamento dopo l'eliminazione di un carattere**

   Quando questa opzione è selezionata, IntelliSense visualizza automaticamente un elenco di completamento rilevante quando si iniziano a eliminare caratteri nel codice.

- **Mostra simboli in spazi dei nomi non aperti**

   Quando questa opzione è selezionata, IntelliSense visualizza automaticamente l'elenco di completamento quando si inizia a digitare e include gli elementi dagli spazi dei nomi non aperti.
