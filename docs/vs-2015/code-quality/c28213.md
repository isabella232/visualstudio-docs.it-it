# <a name="c28213"></a>C28213
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso C28213: l' `_Use_decl_annotations_` annotazione deve essere utilizzata per fare riferimento, senza alcuna modifica, a una dichiarazione precedente. \<why>  
  
 Questo avviso viene segnalato quando viene fatto riferimento a una dichiarazione precedente senza l' `_Use_decl_annotations_` annotazione richiesta.
