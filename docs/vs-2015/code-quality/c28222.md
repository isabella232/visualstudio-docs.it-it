# <a name="c28222"></a>C28222
[!INCLUDE[vs2017banner](../includes/vs2017banner.md)]

avviso 28222: \_ Sì \_ , \_ No \_ o \_ probabilmente \_ previsto per l'annotazione  
  
 Questo avviso indica che un parametro di un'annotazione deve essere uno dei simboli `_Yes_` , `_No_` o e che `_Maybe_` è stato rilevato un altro simbolo. Questo indica in genere una macro di annotazione codificata in modo errato.
