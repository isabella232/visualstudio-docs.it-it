# <a name="side-by-side-comparison-of-git-and-team-explorer"></a>Confronto side-by-side tra Git e Team Explorer

È stata avviata la prima versione di una nuova esperienza Git in Visual Studio 2019 [versione 16.8.](/visualstudio/releases/2019/release-notes/) Questa nuova esperienza consente di ridurre il cambio di contesto con una semplice **finestra delle** modifiche git che include attività Git comuni. Include anche una finestra del **repository Git** a livello di schermo per operazioni Git più avanzate, ad esempio la gestione dei rami e l'esplorazione dei repository.

Se si usa Team Explorer, ecco una guida dettagliata che illustra come usare la nuova esperienza Git.

> [!NOTE]
> La sezione seguente include screenshot ridimensionati in base alle colonne di una tabella. Fare clic su ogni screenshot per visualizzare una versione più grande. Se si usa un dispositivo touchscreen, toccare ogni screenshot per visualizzarlo in una versione più grande.
