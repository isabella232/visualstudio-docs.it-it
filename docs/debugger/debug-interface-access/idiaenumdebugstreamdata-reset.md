# <a name="idiaenumdebugstreamdatareset"></a>IDiaEnumDebugStreamData::Reset
Reimposta l'inizio di una sequenza enumerata.

## <a name="syntax"></a>Sintassi

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>Valore restituito
 Restituisce `S_OK`.
