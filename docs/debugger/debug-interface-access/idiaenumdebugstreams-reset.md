# <a name="idiaenumdebugstreamsreset"></a>IDiaEnumDebugStreams::Reset
Riporta all'inizio la sequenza di enumerazione.

## <a name="syntax"></a>Sintassi

```C++
HRESULT Reset ();
```

## <a name="return-value"></a>Valore restituito
 Restituisce `S_OK`.
