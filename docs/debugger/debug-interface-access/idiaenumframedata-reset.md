# <a name="idiaenumframedatareset"></a>IDiaEnumFrameData::Reset
Riporta all'inizio la sequenza di enumerazione.

## <a name="syntax"></a>Sintassi

```
HRESULT Reset();
```

## <a name="return-value"></a>Valore restituito
 Restituisce `S_OK`.
