# <a name="idiaenumdebugstreamsgetcount"></a>IDiaEnumDebugStreams::get_Count
Recupera il numero di flussi di debug.

## <a name="syntax"></a>Sintassi


#### <a name="parameters"></a>Parametri
