# <a name="ca0503"></a>CA0503
La proprietà è deprecata. Usa la proprietà sostitutiva

L'avviso CA0503 viene generato quando una proprietà di analisi del codice deprecata contiene un valore nel file di configurazione della compilazione o del progetto. Per risolvere il problema, utilizzare la proprietà specificata che la sostituisce.
