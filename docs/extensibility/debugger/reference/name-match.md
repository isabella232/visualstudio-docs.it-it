# <a name="name_match"></a>NAME_MATCH
Seleziona l'opzione di distinzione tra maiuscole e minuscole per i nomi corrispondenti.

## <a name="syntax"></a>Sintassi

```cpp
typedef enum { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
} NAME_MATCH;
```

```csharp
public enum NameMatchOptions { 
   nmNone            = 0,
   nmCaseSensitive   = 1,
   nmCaseInsensitive = 2
}
```

## <a name="fields"></a>Campi
 `nmNone`\
 Non Ã¨ stata specificata alcuna opzione.

 `nmCaseSensitive`\
 Indica che per i nomi di cui trovare una corrispondenza viene fatto distinzione tra maiuscole e minuscole.

 `nmCaseInsensitive`\
 Indica che i nomi di cui trovare una corrispondenza non supportano la distinzione tra maiuscole e minuscole.
