# <a name="ad_process_id"></a>AD_PROCESS_ID
Specifica l'ID processo, che può essere un ID di sistema o un GUID.

## <a name="syntax"></a>Sintassi

```cpp
typedef struct _AD_PROCESS_ID {
    AD_PROCESS_ID_TYPE ProcessIdType;
    union {
        DWORD dwProcessId; 
        GUID  guidProcessId; 
        DWORD dwUnused; 
    } ProcessId;
} AD_PROCESS_ID;
```

```csharp
public struct AD_PROCESS_ID {
    AD_PROCESS_ID_TYPE ProcessIdType;
    DWORD              dwProcessId; 
    GUID               guidProcessId; 
    DWORD              dwUnused; 
};
```

## <a name="members"></a>Members
`ProcessIdType`\
Valore [dell'enumerazione AD_PROCESS_ID_TYPE](../../../extensibility/debugger/reference/ad-process-id-type.md) che specifica come interpretare l'unione o, per il codice gestito, a quale membro `ProcessId` della struttura accedere.

`dwProcessId`\
ID del processo come valore del sistema.

`guidProcessId`\
ID processo come GUID.

dwUnused Padding.
