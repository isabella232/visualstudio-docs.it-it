# <a name="idebugprogramhost2gethostid"></a>IDebugProgramHost2::GetHostId
Ottiene l'identificatore di processo del processo che ospita questo programma.

## <a name="syntax"></a>Sintassi

```cpp
HRESULT GetHostId( 
   AD_PROCESS_ID* pdwId
);
```

```csharp
int GetHostId( 
   AD_PROCESS_ID[] pdwId
);
```

## <a name="parameters"></a>Parametri
`pdwId`\
[in, out] Struttura [AD_PROCESS_ID](../../../extensibility/debugger/reference/ad-process-id.md) che viene compilata con le informazioni sull'identificatore del processo.

## <a name="return-value"></a>Valore restituito
 In caso di esito positivo, restituisce `S_OK`; in caso contrario, restituisce un codice errore.
