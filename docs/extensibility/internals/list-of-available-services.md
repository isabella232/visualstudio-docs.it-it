# <a name="list-of-available-services"></a>Elenco di servizi disponibili

[!INCLUDE[vsprvs](../../code-quality/includes/vsprvs_md.md)]e l'SDK Visual Studio supportano i servizi seguenti. Alcuni pacchetti offrono i propri servizi che non sono elencati qui, ad esempio i servizi di linguaggio non hanno un singolo GUID del servizio. È necessario utilizzare il nome della lingua per trovare il GUID del servizio di linguaggio nel Registro di sistema.

Usare i GUID del servizio elencati qui o ottenuti da un'altra origine (ad esempio, i servizi di linguaggio) per ottenere l'interfaccia o le interfacce principali visualizzate con ogni servizio.
